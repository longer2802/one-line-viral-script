# Breakfast Life Studio v1.0

黃俊榮 Longer 專屬「早餐店人生觀察室 AI 內容總監」。

輸入早餐店素材、故事、觀點或新品資訊，先拆解五項觀察，再產出四種可拍攝、可修改的內容腳本。

## 公開網站

https://longer2802.github.io/one-line-viral-script/

## 啟動

### 在 Mac 電腦使用

雙擊 `啟動網站.command`。終端機會顯示：

- 電腦網址：`http://localhost:4173`
- 手機網址：例如 `http://192.168.1.20:4173`

### 在手機使用

1. 電腦與手機連接同一個 Wi-Fi。
2. 在電腦雙擊 `啟動網站.command`。
3. 將終端機顯示的「手機開啟」網址輸入手機 Safari。
4. 使用期間不要關閉電腦上的終端機視窗。

`localhost` 只能在啟動網站的那台電腦使用，不能直接給手機開啟。

若要讓任何人從外網開啟，需將專案部署到公開網站服務，例如 Cloudflare Pages、Netlify 或 GitHub Pages。

## 功能

- 可貼上最多 1,000 字的完整素材
- 人性、人生、創業、品牌、客戶五項觀察分析
- 探店版、人生觀察室版、產品介紹版、高互動留言版
- 每版包含標題、Hook、分鏡、口播、字幕、CTA
- 所有生成文字皆可直接點擊修改
- 複製單區、單版或全部修改後的內容
- 桌機與手機響應式版面
- 生成引擎完全在瀏覽器端運行
