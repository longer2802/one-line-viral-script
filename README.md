# 早餐店人生觀察室 AI 腳本生成器 V2

黃俊榮 Longer 專屬內容總監。

輸入早餐店素材、故事、觀點或新品資訊，先拆解五項觀察，再依指定風格產出完整短影音與社群內容。

## 公開網站

https://longer2802.github.io/one-line-viral-script/

## 啟動

### 在 Mac 電腦使用

雙擊 `啟動網站.command`。終端機會顯示：

- 電腦網址：`http://localhost:4173`
- 手機網址：例如 `http://192.168.1.20:4173`

### 在手機使用

1. 電腦與手機連接同一個 Wi-Fi。
2. 在電腦雙擊 `啟動網站.command`。
3. 將終端機顯示的「手機開啟」網址輸入手機 Safari。
4. 使用期間不要關閉電腦上的終端機視窗。

`localhost` 只能在啟動網站的那台電腦使用，不能直接給手機開啟。

若要讓任何人從外網開啟，需將專案部署到公開網站服務，例如 Cloudflare Pages、Netlify 或 GitHub Pages。

## 功能

- 可貼上最多 1,000 字的完整素材
- 人性、人生、創業、品牌、客戶五項觀察分析
- 自動辨識八種素材分類
- 探店、人生觀察室、新品上市、都市傳說、創業故事、高互動留言六種風格
- 可單選風格或全部生成
- 每版包含標題、Hook、30 秒腳本、60 秒腳本、分鏡、字幕、CTA
- 自動生成 IG、Facebook、Threads 貼文
- 六項爆款潛力評分、優點與可加強方向
- 所有生成文字皆可直接點擊修改
- 複製單區、單版或全部修改後的內容
- 桌機與手機響應式版面
- 生成引擎完全在瀏覽器端運行
