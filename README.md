# 俊榮 Longer IG 爆款腳本導演

黃俊榮 Longer 專屬 IG 短影音導演、爆款腳本企劃師與心理學文案工具。

不是介紹食物，而是透過食物講故事、透過店家講人生、透過創業講思維。

## 公開網站

https://longer2802.github.io/one-line-viral-script/

## 啟動

### 在 Mac 電腦使用

雙擊 `啟動網站.command`。終端機會顯示：

- 電腦網址：`http://localhost:4173`
- 手機網址：例如 `http://192.168.1.20:4173`

### 在手機使用

1. 電腦與手機連接同一個 Wi-Fi。
2. 在電腦雙擊 `啟動網站.command`。
3. 將終端機顯示的「手機開啟」網址輸入手機 Safari。
4. 使用期間不要關閉電腦上的終端機視窗。

`localhost` 只能在啟動網站的那台電腦使用，不能直接給手機開啟。

若要讓任何人從外網開啟，需將專案部署到公開網站服務，例如 Cloudflare Pages、Netlify 或 GitHub Pages。

## 功能

- 輸入店名、美食、人物習慣或店家故事
- 腳本優先順序固定為情緒、故事、觀點、美食、店家資訊
- 產出反差、好奇、情緒型標題與前 3 秒 Hook
- 60 秒完整腳本，每句字幕不超過 15 字
- 五鏡頭拍攝企劃，包含畫面、口播、字幕與秒數
- 三組封面標題、三組留言 CTA、一組置頂留言
- 自動選擇創業、財富、人生、家庭、品牌或服務思維
- 所有生成文字皆可直接點擊修改
- 複製單區、單版或全部修改後的內容
- 桌機與手機響應式版面
- 生成引擎完全在瀏覽器端運行
